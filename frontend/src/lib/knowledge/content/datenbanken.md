# Datenbanken (SQL)

## Was ist SQL?

SQL ist die Sprache, um mit Datenbanken zu sprechen. Die wichtigste Abfrage ist `SELECT`.

```sql
SELECT name, alter FROM schueler WHERE alter > 14;
```

## Primärschlüssel (Primary Key)

Ein Feld, das jeden Datensatz in einer Tabelle eindeutig identifiziert (z.B. eine ID). Es darf niemals doppelt vorkommen.
