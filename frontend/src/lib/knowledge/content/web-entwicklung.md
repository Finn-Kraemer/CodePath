# Web-Entwicklung

## HTML Grundlagen

HTML strukturiert die Webseite. Alles wird in "Tags" geschrieben.

```html
<h1>Große Überschrift</h1>
<p>Ein Textabsatz</p>
<a href="url">Ein Link</a>
```

## CSS (Styling)

CSS bestimmt das Aussehen (Farben, Abstände, Schriftarten).

```css
h1 {
    color: blue;
    font-size: 24px;
}
```
