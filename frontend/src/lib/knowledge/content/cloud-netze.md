# Cloud & Netzwerke

## Was ist das Internet?

Das Internet ist im Grunde ein riesiges Netzwerk aus Computern, die durch Kabel (oft Glasfaser tief im Ozean) miteinander verbunden sind. Wenn du eine Webseite aufrufst, sendest du eine Anfrage (Request) an einen anderen Computer (Server) irgendwo auf der Welt.

## IP-Adressen

Jedes Gerät im Internet braucht eine Adresse, damit Datenpakete wissen, wohin sie müssen.
- **IPv4:** Die klassische Variante (z. B. `192.168.1.1`). Da diese Adressen langsam ausgehen, gibt es...
- **IPv6:** Die neue, viel längere Variante (z. B. `2001:0db8:85a3:0000:0000:8a2e:0370:7334`).

## DNS (Domain Name System)

Menschen können sich Namen wie `google.com` besser merken als IP-Adressen. Das DNS ist wie das Telefonbuch des Internets. Es übersetzt den Namen, den du eintippst, in die IP-Adresse des Servers.

## Was ist "Die Cloud"?

"There is no cloud, it's just someone else's computer." 
Wenn du Daten "in der Cloud" speicherst, speicherst du sie einfach auf riesigen Server-Farmen von Unternehmen wie Amazon (AWS), Microsoft (Azure) oder Google (GCP).

**Vorteile:**
- Man muss keine eigenen Server kaufen und warten.
- Man kann flexibel Speicherplatz oder Rechenleistung hinzubuchen (Skalierbarkeit).
- Die Daten sind meist gegen Ausfälle gesichert (Backups).
